## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|

| T-04-14 | Elevation of Privilege | runtime service account | mitigate | Grant only roles/secretmanager.secretAccessor (least privilege) for the referenced secret |
| T-04-15 | Denial of Service | endpoint flooding | mitigate | max-instances 3-5 on both functions bounds cost/blast-radius (D-11); accepted residual: no per-request rate limit |
| T-04-16 | Tampering | grace flag misconfiguration | mitigate | REQUIRE_SIGNATURE documented default false during P4->P5 grace; flip-to-true is an explicit post-Phase-5 step (D-02) — prevents accidentally breaking every shipped exe |
| T-04-SC | Tampering | dependency installs | accept | No package installs in this plan (doc + Console ops only); requirements.txt unchanged |
</threat_model>
