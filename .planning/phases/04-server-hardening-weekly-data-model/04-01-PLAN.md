## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-01 | Spoofing | verify_signature() (forged/unsigned curl) | mitigate | HMAC-SHA256 over canonical JSON; verify_signature returns False for missing/invalid sig (enforcement decision lands in Plan 02) |
| T-04-02 | Tampering | canonical_message() (delimiter injection) | mitigate | Canonical JSON (sort_keys, fixed separators) — never a raw `\|` join; machine_id free-text cannot collide field boundaries |
| T-04-03 | Tampering | verify_signature() (signature lifted onto different identity/score) | mitigate | machine_id + initials + score all bound into the signed payload (D-03); helper test asserts cross-identity verify fails |
| T-04-04 | Information Disclosure | verify_signature() (timing oracle on sig compare) | mitigate | hmac.compare_digest constant-time compare; never `==` |
| T-04-05 | Information Disclosure | leaderboard_crypto module (secret in source/logs) | mitigate | Secret read from os.environ at call time; never a module constant, never printed/logged |
